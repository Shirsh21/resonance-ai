# ResonanceAI

ResonanceAI is a real-time voice-to-voice conversational AI agent.

The goal of this project is to build a low-latency AI conversation system that supports live microphone input, real-time speech-to-text, streaming LLM responses, text-to-speech output, and interruption handling.

## Planned Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Node.js
- OpenAI API
- Deepgram Speech-to-Text
- Deepgram or ElevenLabs Text-to-Speech
- WebSockets
- Web Audio API

## Planned Features

- Text-based AI chat
- Streaming AI responses
- Live microphone capture
- Real-time transcription
- Voice output
- Barge-in interruption handling
- Latency tracking dashboard

## Current Status

Day 1: Project initialized with Next.js, TypeScript, and Tailwind CSS.
